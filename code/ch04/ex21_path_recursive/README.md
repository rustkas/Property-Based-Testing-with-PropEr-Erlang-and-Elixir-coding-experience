An OTP library

Create new OTP lib
-----
    $ rebar3 new lib ex21_path_ recursive && cd ex21_path_ recursive

Add information to rebar.config


Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper generators


Format
-----
    $ rebar3 format


PropEr test
-----
    $ rebar3 proper -n 5
    $ rebar3 proper
