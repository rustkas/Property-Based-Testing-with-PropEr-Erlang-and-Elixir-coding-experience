ex01_dupes
=====

An OTP library


Create new OTP lib
-----
    $ rebar3 new lib ex03_collect2 && cd ex03_collect2

Add information to rebar.config


Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper generators


Format
-----
    $ rebar3 format


PropEr test
-----
    $ rebar3 proper
	
	$ rebar3 get-deps && rebar3 proper
