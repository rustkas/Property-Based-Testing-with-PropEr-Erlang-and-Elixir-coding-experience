An OTP library


Create new OTP lib
-----
    $ rebar3 new lib ex06_union && cd ex06_union

Add information to rebar.config


Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper generators


Format
-----
    $ rebar3 format


PropEr test
-----
    $ rebar3 proper
    $ rebar3 get-deps && rebar3 proper
