ex06_thinking
=====

An OTP library

Create new OTP lib
-----
    $ rebar3 new lib ex06_thinking && cd ex06_thinking

Add information to rebar.config
-----

Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper thinking

	
Format
-----
    $ rebar3 format	


PropEr test
-----
    $ rebar3 proper -m prop_thinking
